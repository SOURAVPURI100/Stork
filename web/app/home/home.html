About OneDataShare

OneDataShare is a service which aims to make moving files and data easy for everyone. With enhanced data placement and management features such as end-to-end throughput optimization, early error detection and recovery, and responsive metadata retrieval, we hope to provide a convenient and easy-to-use tool for a broad range of data-intensive cloud computing applications.

The scheduler component underlying the system is open source. We invite developers to contribute patches and new features through pull requests to our GitHub repository: https://github.com/didclab/stork
